General Reports

 

Under this option are displayed the following links. These links can be clicked to view reports:
List of Enabled Companies: This link will open the page that contains a report-listing of all the companies that are currently enabled. In this report following information about the company shall be displayed:
  • Account
  • Company
  • Company Manager Name
  • Email
  • Phone
  • Total User Accounts

Following radio buttons shall be displayed , which shall be selected pertaining the needs of the report.

  • List Companies nearing expiry date
  • List all enabled Companies
  • Number of days to expire
List of Disabled Companies: This link will open the page that contains a report-listing of all the companies that are currently disabled.In this report following information about the company shall be displayed:
  • Account
  • Company
  • Company Manager Name
  • Email
  • Phone
  • Total User Accounts
Leadeship/Management Analysis: It will outline the individual’s leadership or management analysis qualities.
Skill Summary Report: It will outline the various skills of all the users of aparticular group.

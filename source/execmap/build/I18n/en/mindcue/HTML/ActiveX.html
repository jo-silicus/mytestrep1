Please follow undergiven steps for enabling the ActiveX controls:
a) In the browser, go to the Tools --> Internet Options... menu.
b) Select the Security tab.
c) Select zone (Internet/Intranet)
d) Click on the button Custom Level...
    This will display the Security Settings window
e) Go to Initialize and script ActiveX controls not marked as safe and
    select the Enable option
f) Go to Run ActiveX controls and Plug-ins and select the Enable option
g) Press Ok button to save the settings on Security Settings window
h) Press Ok button to save the settings on Internet Options window
i) Please close the browser window and re-login.

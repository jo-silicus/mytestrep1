Global Settings
The Global Settings page contains the following editable fields whereby the behaviour of the SmartSuite application may be manipulated

Default Language : This is the default language to be used in the application.
Global Manager User id: This is the user identification that must be filled in the 'User Id' field in the login screen by the Gloabal Manager.
Global Manager password(max length 8 chars): This is the password that must be filled in the 'Password' field in the login screen by the Global Manager.
Exchange Server Name : This is the name of the exchange server through which data will be transfered.
Exchange Server Port Number : This is the port number of the exchange server through which data will be transfered.
Exchange Server User ID: This is the user id of the exchange server through which data will be transfered.
Exchange Server Password: This is the password of the exchange server through which data will be transfered.

Account Management

This page deals with the the various groups of users (that have been registered through your company) you will be managing as a Company Manager/Group Manager. Here you can perform the following actions:

Select a group: Click on the option button beside the group name to select that group.
View Users comprising a group: Clicking on View Users will take you to the screen where the users comprising that particular group are displayed.
Edit an existing group's profile: Clicking on the 'Edit' button will bring up the screen through which an existing (selected) group account's data may be edited or changed.
View an existing group's profile: Clicking on the group will bring up the screen through which an existing (selected) group's profile, as stored in the databases, is displayed.

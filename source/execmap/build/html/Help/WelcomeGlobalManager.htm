Welcome Global Manager
This is the welcome screen displayed when you successfully login as an Global Manager.
Here you are presented with a number of options:

Global Settings: This option will take you to the screen where you will be able to manipulate the various global settings currently saved in the application.
Add Cobrand Image: This option will allow you to add image into the system and this image will act as a logo for the respective account.
Account Management: This option will take you to the screen where you will be presented with the facilities to manage the accounts(i.e to add a company,edit a company, view a company, add a group, edit a group, view a group, add a company user, edit a company user, view a company user) registered with the system.
Upload File for User Creation: You can automatically establish user accounts by uploading an XML file into the System. You must have a valid corporate account and a valid group established before you can import individual users. First, you must create and XML file similar to this example Sample File. Then click Upload for User Creation. Browse to the file you have created and then click Upload.
Generate Reports: Under this option are displayed the following links. These links can be clicked to view reports:
List of Enabled Companies: This link will open the page that contains a report-listing of all the companies that are currently enabled.
List of Disabled Companies: This link will open the page that contains a report-listing of all the companies that are currently disabled.
Reports: This option will take you to the screen where you will be presented with the options for viewing the Reports.

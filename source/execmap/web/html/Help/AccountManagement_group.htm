Account Management -   Groups

Here, as Global Manager or Company Manager you will be dealing with various groups that comprise a company account. On this page, you have the following actions at your disposal to manage the groups:
Add a group: You may add a group by pressing this button. This button will take you to a page where you may enter the information regarding the new group and submit it.
Edit a group: Here you will be taken to a page where you will be able to edit the selected group's profile. The fields in this page are the same as those in the page for 'Adding' a group.
View a group: You may select the group to view the group's profile.
Enable / Disable: Click on the 'Enable/Disable' button to enable or disable a currently selected group. If the selected group is enabled ,then it will become disabled anf if the selected group is disabled , then it will become enabled.
Delete Disabled Group: This button should be pressed in order to delete the disabled groups. After clicking this button, you can choose to delete all the disabled groups or can choose to delete the selected group only.
Move all group's users to: This button should be pressed in order to move all the users of the selected group into the group whose name is displayed in the drop down list on the right. You may select any group's name from drop down list.
View User : Clicking on View Users will take you to the screen where the users comprising that particular group are displayed.
